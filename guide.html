1. Kurulum

Proje bağımlılıklarını yüklemek için:

composer install

.env dosyasını oluşturun:

DB_HOST=localhost
DB_NAME=seaofsea_db
DB_USER=root
DB_PASS=password
JWT_SECRET=your_secret_key

2. Projeyi Başlatma

PHP'nin dahili sunucusuyla geliştirme ortamında çalıştırabilirsiniz:

php -S localhost:8000

3. Kullanım

3.1 Login Endpoint

Kullanıcı girişi için JWT Token üreten endpoint:

POST /api/login

Gereken Veriler:

{
    "email": "test@example.com",
    "password": "your_password"
}

3.2 Korumalı Endpoint

JWT doğrulaması isteyen endpoint:

GET /api/protected

Header:

Authorization: Bearer {TOKEN}

5. Ek Bilgiler

Flutter kullanıyorsanız CORS ayarlarını doğru yapılandırın:

header("Access-Control-Allow-Origin: *");
header("Access-Control-Allow-Methods: GET, POST, PUT, DELETE");
header("Access-Control-Allow-Headers: Content-Type, Authorization");
